✅ LeetCode — Count Primes (LC 204)

🔹 Problem Statement (Simplified)
Given an integer n, count all prime numbers strictly less than n.

🔹 Example

Input: 10
Output: 4  (2, 3, 5, 7)

Input: 0
Output: 0

Input: 1
Output: 0


🔹 Interview-Preferred Approach: Sieve of Eratosthenes
Why Sieve?

- Eliminates multiples in 1 pass
- Much faster than checking each number individually
- Standard optimized approach

🔹 Algorithm

1.Create a boolean array prime[0…n-1], mark all true
2.Set 0 and 1 as false
3.For i = 2 to √n, if prime[i] == true:
   -Mark all multiples j = i*i to n as false
4.Count all true values

CODE 
class Solution {
    public int countPrimes(int n) {
        if (n < 2) return 0;

        boolean[] prime = new boolean[n];
        for (int i = 2; i < n; i++) {
            prime[i] = true;
        }

        for (int i = 2; i * i < n; i++) {
            if (prime[i]) {
                for (int j = i * i; j < n; j += i) {
                    prime[j] = false;
                }
            }
        }

        int count = 0;
        for (int i = 2; i < n; i++) {
            if (prime[i]) count++;
        }
        return count;
    }
}

🔹 Time & Space Complexity
Time: O(n log log n)
Space: O(n)

🔥 Interview Tip
If asked:
Why start marking from i * i?
Answer:
Smaller multiples are already marked by previous primes, so we start from i².
