✅ LeetCode — Pow(x, n) (LC 50)

🔹 Problem Statement (Simplified)
Compute xⁿ (power of x to n).
n can be negative, zero, or positive.
Result should be efficient for large n.

🔹 Example

Input: x = 2.0, n = 10
Output: 1024.0

Input: x = 2.0, n = -2
Output: 0.25


🔹 Interview-Preferred Approach: Fast Exponentiation (Binary Exponentiation)
Why this?
-Avoids O(n) multiplications
-Works in O(log n)
-Clean recursive or iterative logic

🔹 Algorithm

1.If n < 0 → convert to positive and use 1/x
2.Base case: n == 0 → return 1
3.Recursively compute half = pow(x, n/2)
4.If n even → return half * half
5.If n odd → return half * half * x

class Solution {
    public double myPow(double x, int n) {
        if (n == 0) return 1;
        if (n < 0) return 1 / myPow(x, -n);
        double half = myPow(x, n / 2);
        return (n % 2 == 0) ? half * half : half * half * x;
    }
}

🔹 Time & Space Complexity
Time: O(log n)
Space: O(log n) recursion stack

🔥 Interview Tip
If asked:
Why not multiply x n times?
Answer:
That takes O(n). Fast exponentiation reduces it to O(log n), much faster for large n.
