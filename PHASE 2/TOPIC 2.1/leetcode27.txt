✅ LeetCode — Remove Element (LC 27)

🔹 Problem Statement (Simplified)
Given an array nums and a value val, remove all occurrences of val in-place and return the new length.
The order of remaining elements does not matter.

🔹 Example
Input:  nums = [3,2,2,3], val = 3
Output: 2, nums = [2,2,_ ,_]
Input:  nums = [0,1,2,2,3,0,4,2], val = 2
Output: 5, nums = [0,1,3,0,4,_ ,_ ,_]


🔹 Interview-Preferred Approach: Two Pointer (Overwrite Valid Elements)
Why this?
No extra array
In-place update
1-pass efficient

🔹 Algorithm
Use pointer k = 0 to track valid index
Loop i = 0 → n-1
If nums[i] != val → copy nums[k++] = nums[i]
Continue loop
Return k as new length

🔹 Java Code (BEST ANSWER)

class Solution {
    public int removeElement(int[] nums, int val) {
        int k = 0;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != val) {
                nums[k++] = nums[i];
            }
        }
        return k;
    }
}


⏱ Time & Space Complexity
Time: O(n)
Space: O(1) auxiliary

🔥 Interview Tip
If asked:
Why not delete elements directly?
Answer:
Deleting causes shifting → O(n²) worst case. Overwriting avoids shifts → O(n) optimal.
